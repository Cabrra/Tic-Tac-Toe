Tic Tac Toe
===========

## Features
+ Blocking AI (not random).
+ Player chooses "X" or "O".
+ Randomized who goes first.
+ Board drawn in the center

## Built With

* [Visual Studio](https://visualstudio.microsoft.com/)	- C++ development
* Run with Windows CMD
